Using these controls

 The controls on this screen allow the dbo user the ability to select which permissions users have for specific tables within Easysoft Data Access for ISAM.
Click Grant All to grant full access to all tables.
Click Revoke All to revoke access from all tables.
Click Submit Selection to apply the displayed access rights for the specified user.

Notes
  • Choose the public user if you want to make tables accessible to all users.
